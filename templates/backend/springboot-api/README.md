# Spring Boot API Template

一个可直接复制的新项目后端模板，适用于：

- Spring Boot 单体 API
- Linux + systemd 部署
- 外置 `application-prod.yml`
- Maven 打包
- 支持后续接入自动化部署

## Included

- `application-prod.yml.template`：生产环境配置模板
- `systemd/app.service.template`：systemd 服务模板
- `pom-deploy-profile.xml`：Maven 部署 profile 片段
- 基础目录约定与部署说明
